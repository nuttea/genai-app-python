# Cursor Custom Slash Commands

This directory contains custom slash commands for Cursor AI.

According to the [Cursor documentation](https://cursor.com/docs/agent/chat/commands), commands are **plain Markdown files** (`.md`).

## Available Commands

- `/lint-commit-push` - Complete workflow (format, lint, commit, push)
- `/format-only` - Format code without committing
- `/quick-push` - Quick push with auto-generated message
- `/organize-docs` - Organize documentation and scripts into proper directories

## Usage

1. Open Cursor Agent: `Cmd/Ctrl + L`
2. Type `/` to see commands
3. Select your command
